# vuepress лЎң GitHub pages м—җ мӮ¬мқҙнҠё нҳёмҠӨнҢ… н•ҳлҠ” л°©лІ•

мқҙм „м—җлҠ” GitHub pages + м§ҖнӮ¬ н…Ңл§ҲлҘј мқҙмҡ©н•ҙм„ң лё”лЎңк·ёлҘј л§Ңл“Өм—Ҳм—ҲлҠ”лҚ°, мғҲлЎң лё”лЎңк·ёлҘј л§Ңл“ӨкІҢ лҗҳл©ҙм„ң м°ҫлӢӨк°Җ м•ҢкІҢлҗң vuepressлЎң GitHub pagesм—җ мӮ¬мқҙнҠё нҳёмҠӨнҢ… н•ҳлҠ” л°©лІ•м—җ лҢҖн•ҙм„ң нҸ¬мҠӨнҢ…н•ҳл Өкі  н•ңлӢӨ.

лӢӨлҘё мӮ¬мқҙнҠём—җм„ңлҸ„ vuepress лҘј мқҙмҡ©н•ҙм„ң github pagesм—җ нҳёмҠӨнҢ… н•ҳлҠ” кёҖл“Өмқҙ л§Һм§Җл§Ң мҙҲліҙмһҗ лҲҲ лҶ’мқҙм—җ л§һм¶”м–ҙ м°Ёк·јм°Ёк·ј м •лҰ¬лҗң кёҖмқҖ м—ҶлҠ” кІғ к°ҷм•„ мғҒм„ёнһҲ м •лҰ¬н•ҳл Өкі  н•ңлӢӨ.

vuepressмҷҖ github pages лҘј мқҙмҡ©н•ҳл©ҙ мң„мҷҖ к°ҷмқҖ лё”лЎңк·ёлҘј л§Ңл“Ө мҲҳ мһҲлӢӨ.

л””мһҗмқёлҸ„ көүмһҘнһҲ к№”лҒ”н•ҳкі  IT м •ліҙм„ұ лё”лЎңк·ёлЎңлҠ” кө°лҚ”лҚ”кё°к°Җ м—ҶлҠ” кІғмқҙлқј мғқк°Ғн•ҙм„ң мқҙ мҠӨнғҖмқјмқ„ мұ„нғқн•ҳкІҢ лҗҳм—ҲлӢӨ.

мқҙ кёҖм—җм„ңлҠ” github pagesмҷҖ vuepressмқҳ к№ҠмқҖ м–ҳкё°лҠ” л’ӨлЎңн•ҳкі  github pagesм—җ vuepressлҘј мқҙмҡ©н•ҙм„ң лё”лЎңк·ё мӮ¬мқҙнҠёлҘј нҳёмҠӨнҢ… н•ҳлҠ” л°©лІ•м—җл§Ң мҙҲм җмқ„ л§һм¶”м–ҙ нҸ¬мҠӨнҢ…н•ҳл Өкі  н•ңлӢӨ.

# github м ҖмһҘмҶҢ мғқм„ұ

TILмқҙлқјлҠ” м ҖмһҘмҶҢлҘј мғқм„ұн•ҙмӨҖлӢӨ. м ҖмһҘмҶҢмқҳ мқҙлҰ„мқҖ м–ҙл–Ө кІғмқҙм–ҙлҸ„ мғҒкҙҖ м—ҶлӢӨ.

м ҖмһҘмҶҢлҘј мғқм„ұн•ҙмЈјм—ҲлӢӨл©ҙ н•ҙлӢ№ м ҖмһҘмҶҢм—җ vuepressлҘј м„Өм№ҳн•ҙм•јн•ңлӢӨ.

# vuepress м„Өм№ҳ

vuepressлҘј м„Өм№ҳн•ҳлҠ” л°©лІ•мқҖ 2к°Җм§ҖлЎң лӮҳлүңлӢӨ. ліҙмқјлҹ¬н”Ңл ҲмқҙнҠёлҘј к°Җм§Җкі  мһҲлҠ” нҢЁнӮӨм§ҖлҘј м„Өм№ҳн•ҳкұ°лӮҳ м§Ғм ‘ мҲҳлҸҷмңјлЎң vuepressлҘј м„Өм№ҳн•ҳлҠ” л°©лІ•мқҙлӢӨ.

мқҙлІҲ кёҖм—җм„ңлҠ” vuepressлҘј мҲҳлҸҷмңјлЎң м„Өм№ҳн•ҙм„ң м•„лһҳмҷҖ к°ҷмқҖ кё°ліём Ғмқё нҳ•нғңмқҳ мӮ¬мқҙнҠёлҘј нҳёмҠӨнҢ…н•ҳлҠ” кІғк№Ңм§Җл§Ң лӢӨлЈЁл Өкі  н•ңлӢӨ.

лӢӨмқҢм—җлҠ” github pagesм—җ vuepressмқҳ н…Ңл§ҲмҷҖ мҠӨнғҖмқјмқ„ м Ғмҡ©н•  мҲҳ мһҲлҠ” л°©лІ•мқ„ нҸ¬мҠӨнҢ…н•ҳл Өкі  н•ңлӢӨ.

лӢӨмӢң ліёлЎ мңјлЎң лҸҢм•„мҷҖм„ң

# 1. Npm м„Өм№ҳ

м ҖмһҘмҶҢ лЈЁнҠёл””л үнҶ лҰ¬м—җм„ң м•„лһҳ лӘ…л №м–ҙлҘј мһ…л Ҙн•ңлӢӨ.

vuepressлҘј м„Өм№ҳн•ҳкё° мң„н•ҙм„ңлҠ” yarn, npmмқҙ н•„мҡ”н•ҳкё° л•Ңл¬ёмқҙлӢӨ.

npm init

# 2. vuepress м„Өм№ҳ

npmмқ„ м„Өм№ҳн–ҲмңјлӢҲ vuepressлҘј м„Өм№ҳн•ҙмӨҖлӢӨ.

yarn add -D vuepress

# 3. нҸ¬мҠӨнҠё мһ‘м„ұ

GitHub pages лҘј мӮ¬мҡ©н•ҳлҠ” л°©лІ•м—җлҠ” м—¬лҹ¬к°Җм§Җк°Җ мһҲлҠ”лҚ° лЈЁнҠё л””л үнҶ лҰ¬м—җ docs/ нҸҙлҚ” н•ҳмң„м—җ кёҖмқ„ мһ‘м„ұн•ҳлҠ” л°©мӢқмқ„ м„ нғқн–Ҳкё° л•Ңл¬ём—җ docs нҸҙлҚ”лҘј л§Ңл“Өм–ҙ н•ҳмң„м—җ л§ҲнҒ¬лӢӨмҡҙ л¬ём„ңлҘј мһ‘м„ұн•ҙмӨҖлӢӨ.

mkdir docs && echo '# Hello VuePress' > docs/README.md

README нҢҢмқјмқҖ нҷҲм—җ л…ём¶ңлҗ  кёҖмқ„ мһ‘м„ұн•ҳлҠ” кіімқҙкі 

нҸ¬мҠӨнҢ…мқ„ н•ҳлҠ” кІғмІҳлҹј vue-press.md нҢҢмқјмқ„ н•ҳлӮҳ мғқм„ұн•ҙм„ң м•„лһҳмҷҖ к°ҷмқҙ мһ„мқҳлЎң кёҖмқ„ мһ‘м„ұн•ҙлҙ…лӢҲлӢӨ

# vuepress

# 4. Package.json мһ‘м„ұ

package.json мқҳ script л¶Җл¶„мқ„ м•„лһҳмҷҖ к°ҷмқҙ м¶”к°Җн•ҙмӨҖлӢӨ.

{
  "scripts": {
    "docs:dev": "vuepress dev docs",
    "docs:build": "vuepress build docs"
  }
}

# 5. vuepress м„Өм •

docs н•ҳмң„м—җ .vuepress/config.js нҸҙлҚ”лҘј л§Ңл“Өм–ҙмӨҚлӢҲлӢӨ.

к·ёлҰ¬кі  м•„лһҳ лӮҙмҡ©мқ„ мұ„мӣҢмӨҚлӢҲлӢӨ.

module.exports = {
  title: "Hello VuePress",
  description: "Just playing around",
};

# 6. лЎңм»¬ мӢӨн–ү

yarn docs:dev

vuepress м„Өм№ҳлҠ” лҒқлӮ¬лӢӨ. мқҙм ң лЎңм»¬м—җм„ң vuepressк°Җ м •мғҒм ҒмңјлЎң лҸҷмһ‘н•ҳлҠ”м§Җ н…ҢмҠӨнҠё н•ҙліҙл©ҙ лҗңлӢӨ.

Localhost:8000 мңјлЎң м ‘к·јн•ҳл©ҙ мң„мҷҖ к°ҷмқҖ нҷ”л©ҙмқҙ лңЁлҠ” кІғмқ„ ліј мҲҳ мһҲлӢӨ.

к·ёлҹ¬л©ҙ мқҙм ң GitHub pages м—җ нҳёмҠӨнҢ…мқ„ н•ҙм•јн•ңлӢӨ.

# 7. GitHub pages нҳёмҠӨнҢ…

github pages м—җ нҳёмҠӨнҢ…мқ„ н•ҳл Өл©ҙ м ҖмһҘмҶҢлҘј githubID.github.io лЎң л§Ңл“ңлҠ” л°©лІ•лҸ„ мһҲкі  мң„м—җм„ң л§җн•ң кІғмІҳлҹј docs л””л үнҶ лҰ¬ н•ҳмң„м—җ нҸ¬мҠӨнҢ…мқ„ н•ҳлҠ” л°©лІ•мқҙ мһҲлӢӨ. docs л°©лІ•мқ„ н•ҳл Өл©ҙ лёҢлһңм№ҳлҘј gh-pages м—җ н‘ёмӢңн•ҙм•јн•ңлӢӨ.

мқҙкұҙ к·ёлғҘ githubм—җм„ң л§Ңл“  к·ңм№ҷмқҙлқјкі  мғқк°Ғн•ҳл©ҙ лҗңлӢӨ. мқҳл¬ёмқ„ к°Җм§Ҳ н•„мҡ”к°Җ м—ҶлӢӨ.

git checkout -b gh-pages

git add .

git commit -m "initial commit"

git push origin gh-pages

м ҖмһҘмҶҢмқҳ settings м—җ л“Өм–ҙк°Җл©ҙ мўҢмёЎм—җ pages л©”лүҙк°Җ мһҲлӢӨ м—¬кё° л“Өм–ҙмҳӨл©ҙ github pagesм—җ лҢҖн•ң м •ліҙлҘј нҷ•мқён•  мҲҳ мһҲлӢӨ.

branchм—җ /docs л””л үнҶ лҰ¬лЎң лҗҳм–ҙмһҲм–ҙм•ј м •мғҒ лҸҷмһ‘н•ҳлӢҲ м°ёкі н•ҳмӢңкёё л°”лһҖлӢӨ. (rootлЎң лҗҳм–ҙмһҲмқ„ кІҪмҡ° нҳёмҠӨнҢ…мӢң 404 м—җлҹ¬к°Җ л°ңмғқ)

мқҙм ң к№ғн—ҷм•„мқҙл””.github.io/м ҖмһҘмҶҢмқҙлҰ„ м—җ м ‘к·јн•ҳл©ҙ м•„лһҳмҷҖ к°ҷмқҙ нҳёмҠӨнҢ…мқҙ лҗң кІғмқ„ ліј мҲҳ мһҲлӢӨ.

мң„ нҷ”л©ҙмқҙ нҷҲ нҷ”л©ҙмқҙкі  м•„к№Ң vue-press.md нҸ¬мҠӨнҢ…мқ„ ліҙкі  мӢ¶лӢӨл©ҙ к№ғн—ҷм•„мқҙл””.github.io/м ҖмһҘмҶҢмқҙлҰ„/vue-press лЎң м ‘к·јн•ҳл©ҙ

мқҙл ҮкІҢ нҸ¬мҠӨнҢ…м—җ м ‘к·јн•  мҲҳ мһҲлӢӨ.

мқҙлІҲ нҸ¬мҠӨнҢ…м—җм„ңлҠ” лӢӨлҘё л¶Җл¶„ л№јкі  vuepressлҘј github pages м—җ м–ҙл–»кІҢ нҳёмҠӨнҢ… н•ҳлҠ”м§Җм—җл§Ң лӢӨлЈЁм—Ҳкі , мң„ мқҙлҜём§ҖмІҳлҹј лё”лЎңк·ёлҘј кө¬м„ұн•ҳкё°м—җлҠ” л„Ҳл¬ҙ ліјн’Ҳ м—Ҷкё° л•Ңл¬ём—җ ліҙмқјлҹ¬н”Ңл ҲмқҙнҠёлҘј нҷңмҡ©н•ҳкұ°лӮҳ мҠӨнғҖмқјмқ„ м»ӨмҠӨн…Җн•ҳлҠ” л°©мӢқм—җ лҢҖн•ҙм„ңлҸ„ м¶”к°ҖлЎң лӢӨлЈЁм–ҙм•јн•  кІғ к°ҷлӢӨ.
